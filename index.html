html网页和网页结构

发表于 2019-02-01 | 分类于 Testing

学习目的

  • 认识网页的基本组成和标准
  • 熟悉 HTML 的组成结构

    1、认识网页

  • 网页组成：由文字、图片、按钮、搜索框、视频等元素（HTML 标签）组成，还包含音频、视频、FLASH等

    总结：网页其实就是由HTML标签组成的。

  • 网页标准

    1. 结构标准：（相当于人的身体）HTML → 搭建网页结构
    2. 表现标准：（相当于人的衣服）CSS → 美化网页
    3. 行为标准：（相当于人的动作）Javascript

  • 浏览器

    1. 浏览器是网页运行的平台，常用的浏览器有 IE、火狐（Firefox）、谷歌（Chrome）、Safari 和 Opera等。
    2. 内核（渲染引擎）： 将网页中的标签信息转化为我们人眼能够识别的图文信息。

2、网页结构

  • 什么是 HTML
    1. HTML：Hyper Text Markup Language
    2. 超文本：在网页中能够实现页面跳转的文字（超链接）
    3. 标记：指的是在网页中的 html 标签

  • HTML 结构介绍

    本文是基于 html5 结构去设计网页的。

  • 详细介绍结构中的标签

  1. DOCTYPE 标签：告诉浏览器当前文档的类型，如果不设置 DOCTYPE 那么浏览器在显示过程中可能会出现一种怪异模式。
  2. html 标签：结构中的根标签
  3. head 标签： 可以设置网页样式；可以设置网页标题；设置网页关键字；网页描述信息；js 代码编写。
  4. title 标签： 设置网页标题
  5. body 标签： 指的是网页的主体（我们看到的所有信息）
